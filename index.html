First steps to start work with Git

Gallery page
git init
Инициализация репозитария в рабочей папке
git remote add origin https://github.com/Crazy-Sirius/HW_Lesson_03.git
Подключаем локальный репозиторий а ГитХабу по https протоколу
git add index.html gallery.html images
Индексация изменений в файлах index.html gallery.html и папке images
git commit -m “Комментарий к изменениям”
Коммитим изменения в файлах и папке
git push origin master
Отправка на ГитХаб
